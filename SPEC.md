# ペット体重記録・肥満診断【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Pet Weight Tracker
**バージョン**: 1.1.0
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/pet-weight-tracker/

犬・猫の体重を記録して肥満度をBCS（ボディコンディションスコア）で診断。登録不要・完全無料でご利用いただけます。

## バージョン履歴

### v1.1.0 (2026-06-13)
- **M4**: Chart.js を使用した体重推移グラフの追加（折れ線グラフ・理想体重との比較）
- **M6**: 目標設定・達成度トラッキング機能の追加（目標体重の設定・達成度パーセンテージ表示・目標までの差分表示）
- **M8**: ツールチップ・入力ガイダンス充実（各入力フィールドに title 属性・? アイコン・説明テキストを追加）
- **M10**: 印刷・結果エクスポート機能の追加（window.print() ボタン・clipboard コピー機能・CSV ダウンロード機能）

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: ペット体重記録・肥満診断は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
